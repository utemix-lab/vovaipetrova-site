# Быстрый старт

## Локальный запуск

1. **Импорт данных из core:**
   ```bash
   npm run import
   ```

2. **Сборка сайта:**
   ```bash
   npm run build
   ```

3. **Просмотр:**
   - Откройте `dist/index.html` в браузере
   - Или используйте локальный сервер:
     ```bash
     cd dist
     python -m http.server 8000
     ```
     Затем откройте http://localhost:8000

## Структура

- `data/` - импортированные данные из core
- `dist/` - собранный сайт (для деплоя)
- `src/` - исходники (шаблоны, стили)
