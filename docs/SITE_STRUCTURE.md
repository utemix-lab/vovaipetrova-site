# Структура сайта (витрина)

## Принцип разделения

**Сайт (vovaipetrova-site)** — витрина проекта:
- Классическая структура: меню, разделы, подразделы
- Понятная навигация для пользователей
- Независимая от структуры ядра

**Ядро (vovaipetrova-core)** — для тех, кто хочет заглянуть под капот:
- Техническая документация
- Процессы разработки
- ADR и архитектурные решения

## Структура сайта (витрина)

### Главная (`/`)
- О проекте
- Быстрая навигация по разделам
- CTA (призыв к действию)

### Разделы

1. **Услуги** (`/services`)
   - Видеопродакшн
   - Дизайн
   - Проектирование
   - Автоматизация

2. **Портфолио** (`/portfolio`)
   - Кейсы
   - Проекты
   - Результаты

3. **База знаний** (`/kb`)
   - Статьи
   - Инструменты
   - Гайды

4. **О нас** (`/about`)
   - О проекте
   - Команда
   - Контакты

5. **Блог/Stories** (`/stories`)
   - Новости проекта
   - Эпизоды развития

## Навигация

### Главное меню
- Услуги
- Портфолио
- База знаний
- О нас
- Блог

### Подменю (для разделов с подразделами)
- Услуги → Видео, Дизайн, Проектирование
- Портфолио → Кейсы, Проекты
- База знаний → Статьи, Инструменты

## Компоненты интерфейса

1. **Header** — шапка с логотипом и главным меню
2. **Sidebar** — боковое меню (для разделов с подразделами)
3. **Main Content** — основной контент страницы
4. **Footer** — подвал с контактами и ссылками
5. **Breadcrumbs** — навигационные крошки

## Данные

Структура сайта определяется в `src/config/site-structure.json` (независимо от ядра).

Данные для контента будут подтягиваться из ядра позже, но структура сайта — своя.
