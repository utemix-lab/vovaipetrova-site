## Добавление контента в разделы сайта

### Что сделано

1. **Создан компонент ContentLoader** (`src/core/content-loader.js`)
   - Загрузка данных KB glossary из core
   - Загрузка данных Stories digests из core
   - Рендеринг контента для разных разделов

2. **Интеграция контента в роутер**
   - Автоматическая загрузка контента после рендеринга страницы
   - Поддержка главной страницы (превью услуг и портфолио)
   - Поддержка раздела KB (список терминов)
   - Поддержка раздела Stories (дайджесты)

3. **Улучшения роутера**
   - Добавлены методы `renderKBPage` и `renderStoriesPage`
   - Асинхронная загрузка контента
   - Правильная обработка маршрутов

### Результат

Теперь разделы сайта автоматически наполняются контентом из core:
- Главная страница показывает превью услуг
- Раздел KB показывает список терминов из базы знаний
- Раздел Stories показывает дайджесты эпизодов
